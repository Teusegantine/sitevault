5. Animações Globais

  • Smooth Scrolling: Ativado globalmente na tag HTML.
  • st-fade-up: Classe utilitária (removida no CSS local, mas usa lógica GSAP no main.js) que faz os elementos surgirem de baixo para cima acompanhando o scroll.
  • Hover States: Todos os botões interativos e links mudam de cor suavemente (transition-colors duration-300).
